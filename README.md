# Nexus 🚀

Nexus is a browser-based collaborative code editor that enables developers to write, edit, and collaborate on code in real time. It combines a modern code editor, file management system, live collaboration, and code execution into a clean and responsive interface.

---

## ✨ Features

### 📝 Code Editor
- Monaco Editor integration
- Syntax highlighting
- Multiple language support
- Auto indentation
- Multiple file tabs
- Markdown preview support

### 📁 File Management
- Create files and folders
- Rename files and folders
- Delete files and folders
- Nested folder structure
- File explorer with language-specific icons

### 🤝 Real-Time Collaboration
- Create and join collaboration rooms
- Live code synchronization
- Real-time remote cursors
- Collaborator presence
- Shared collaborative file tree
- Active participant list

### ▶️ Code Execution
- Execute code using Judge0 API
- Multiple programming language support
- Input support
- Output console
- Error handling

### 🎨 Modern UI
- VS Code-inspired interface
- Responsive layout
- Resizable panels
- File tabs
- Context menus
- Clean dark theme

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/nexus.git
cd nexus
```

---

### 2. Install dependencies

#### Frontend

```bash
cd frontend
npm install
```

#### Backend

```bash
cd backend
npm install
```

---

## ▶️ Running the Project

### Start Backend

```bash
cd backend
npm run dev
```

---

### Start Frontend

```bash
cd frontend
npm run dev
```

## 🔄 Collaboration Workflow

1. Create a collaboration room.
2. Share the room ID with teammates.
3. Collaborators join using the room ID.
4. All file edits sync instantly.
5. Remote cursors display each collaborator's position.
6. Collaborators can edit shared files simultaneously.
